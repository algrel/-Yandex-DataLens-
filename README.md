Анализ и визуализация данных с использованием Yandex DataLens, на примере данных Банка России
Данный репозиторий предназначен для хранения программ, преобразующих данные из файлов excel в нужны для визуализации в Yandex Datalens формат. Файлы с "_до" в названии получаются путем несложных (удаление лишних столбцов, листов, смена заголовков приведение даты в нужный формат) преобразований таблиц данных  с сайта Центробанка России. фо_гео получен из "Показатели естественного движения населения. Демография" (Интерактивный дашборд с открытыми данными Росстата по общим демографическим итогам и приросту населения.). После запуска файлов *py будут получены файлы, использованные непосредственно в визуализации.
